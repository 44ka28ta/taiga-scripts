# Taiga Bootstrap Scripts #

Scripts for initial deploy of taiga services.

It installs:

- taiga-back version 1.1.0
- taiga-front version 1.1.1

**WARNING: that scripts are in aplha state**

## Requirements ##

- Ubuntu Server 14.04
- Bash
- Git
